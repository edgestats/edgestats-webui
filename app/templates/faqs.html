How does it works?
EdgeStats client watches edge node log file
Scans edge node logs as they are written
Filters edge node logs relevant to uptime
Sends uptime logs to the EdgeStats server
Web UI displays the edge node uptime stats
Important to note!
An edge node's uptime stats will not be available unless the EdgeStats client was run previously
If an edge node is running but the EdgeStats client is not, the uptime stats are not being collected by EdgeStats
If either the EdgeStats client or edge node is not running, uptime stats are not being collected by EdgeStats
